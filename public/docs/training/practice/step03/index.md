# プルリクエスト作成からマージ

以下のことをやりましょう。

1. Githubで `new pull request` を作成してください。
2. マージ先を `develop` に変更、プルリクエスト作成完了まで行いましょう。  
タイトル、説明に関して、やったことがわかりやすいように記載してください。  
※ 正解はないです。伝わるように意識して記載してください。
3. Githubの操作で `マージ` してください。
