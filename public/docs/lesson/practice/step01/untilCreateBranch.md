# ブランチを作成するまで

ここで行うことは以下のことをやってみましょう。
1. `mainブランチ`をチェックアウトしていることを確認しましょう
2. `developブランチ`をチェックアウトしましょう
3. `developブランチ`から2つ新しいブランチを作成しましょう。  
※ ブランチ名は`feature/check-type`と`feature/add-list-item`にしてください。
