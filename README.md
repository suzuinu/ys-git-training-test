# ys-git-training-fizz-buzz

Gitの使い方の練習のために作成したリポジトリになります。  
こちらのリポジトリは、forkさえすれば壊しても問題ありません。  
もしGit操作に失敗した場合は、再度このリポジトリをForkしてもらえれば0からやり直せます。

## このカリキュラムで学べること

- Gitに関する基礎的な知識
- 実務でのGitの扱う流れ
- Gitを使用するときに使うコマンド
